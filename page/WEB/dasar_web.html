Dasar Website

Apa itu Website?

Website adalah kumpulan halaman yang berisi informasi (teks, gambar, video, dan sebagainya) yang dapat diakses melalui internet menggunakan browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge.

Contoh website:

  • Website berita: https://kompas.com
  • Website sekolah: https://smkkrian1.sch.id
  • Website media sosial: https://facebook.com
Komponen Dasar Website

Website dibangun dari beberapa komponen utama, yaitu:

  1. HTML (HyperText Markup Language)
  2. CSS (Cascading Style Sheets)
  3. JavaScript
Struktur File Website

Dalam membuat website, struktur folder dan file perlu ditata dengan rapi agar mudah dikelola dan dikembangkan. Contoh struktur file sederhana: 

/project-website/
│
├── index.html
├── about.html
├── /css/
│   └── style.css
├── /js/
│   └── script.js
├── /images/
│   └── logo.png
└── /assets/
    └── font.ttf

Struktur ini akan berkembang seiring penggunaan framework atau metode seperti MVC.

Tugas 1: Membuat Landing Page Web Portofolio

Pada tugas ini, kalian diminta untuk membuat sebuah landing page sederhana sebagai portofolio pribadi menggunakan HTML dan CSS.

Instruksi:

  • Buat file index.html untuk menampilkan struktur konten.
  • Buat file style.css untuk mengatur tampilan halaman.
  • Gunakan elemen seperti: header, section, dan footer.
  • Tambahkan informasi seperti nama, foto, deskripsi diri, dan kontak.

Simpan kedua file dalam satu folder dan pastikan tampilan rapi dan responsif.
